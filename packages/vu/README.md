# vu
根据ant-design-vue再次封装


```shell
安装依赖

npm --workspace vu add -D ant-design-vue
npm --workspace vu add -D @ant-design/icons-vue
npm --workspace vu add -D @vueuse/core
npm --workspace vu add -D unplugin-auto-import
npm --workspace vu add -D unplugin-vue-components
npm --workspace vu add -D @types/node
npm --workspace vu add -D sass
npm --workspace vu add -D normalize.css

npm --workspace vu run build
```
